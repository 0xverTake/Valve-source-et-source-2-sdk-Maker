# Valve SDK Generator v2.1.0

## Description
Cet outil permet de générer automatiquement un SDK compatible avec les jeux basés sur les moteurs Source et Source 2 de Valve.

## Installation
1. Extrayez tous les fichiers dans un dossier de votre choix
2. Exécutez ValveSDKMaker.exe en tant qu'administrateur

## Utilisation
1. Lancez d'abord le jeu cible
2. Ouvrez ValveSDKMaker.exe
3. Sélectionnez le jeu cible dans la liste déroulante
4. Cliquez sur "Générer SDK"
5. Attendez la fin du processus

## Configuration avancée
Vous pouvez modifier les paramètres avancés en éditant le fichier config.ini.

## Dépannage
Si vous rencontrez des problèmes :
- Assurez-vous d'exécuter le programme en tant qu'administrateur
- Vérifiez que le jeu cible est en cours d'exécution
- Consultez les logs dans le dossier "logs"
