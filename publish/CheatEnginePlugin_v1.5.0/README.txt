# Plugin Cheat Engine pour Valve Source SDK v1.5.0

## Description
Ce plugin pour Cheat Engine facilite l'analyse et la modification des jeux basés sur les moteurs Source et Source 2 de Valve.

## Installation
1. Assurez-vous que Cheat Engine 7.2 ou supérieur est installé
2. Exécutez install.bat pour installer automatiquement les scripts
3. Redémarrez Cheat Engine

## Installation manuelle
Si l'installation automatique ne fonctionne pas :
1. Ouvrez Cheat Engine
2. Allez dans "Edit" > "Settings" > "Lua Engine"
3. Cliquez sur "Add" et sélectionnez les fichiers .lua de ce dossier

## Fonctionnalités
- CustomSignatures.lua : Signatures personnalisées pour les jeux Source
- SourceTableGenerator.lua : Générateur automatique de tables pour les structures de données Source

## Utilisation
1. Lancez le jeu cible
2. Ouvrez Cheat Engine et attachez-le au processus du jeu
3. Les nouvelles fonctionnalités apparaîtront dans le menu "Table" et "Memory View"
