Installation

SDK Generator

  1. Téléchargez la dernière version du SDK Generator
  2. Extrayez l'archive dans un dossier de votre choix
  3. Exécutez ValveSDKMaker.exe

Cheat Engine Plugin

  1. Téléchargez la dernière version du plugin
  2. Extrayez l'archive
  3. Exécutez install.bat pour installer automatiquement le plugin
  4. Redémarrez Cheat Engine

Utilisation du SDK Generator

Configuration du jeu cible

Par défaut, l'outil est configuré pour cibler Black Mesa. Pour changer le jeu cible, modifiez la constante TARGET_GAME dans le fichier main.cpp :

// Options: blackmesa, hl2, css, gmod, portal, portal2, csgo, dota2, custom
const std::string TARGET_GAME = "blackmesa";

Génération du SDK

  1. Lancez d'abord le jeu cible
  2. Exécutez ValveSDKMaker.exe en tant qu'administrateur
  3. L'outil détectera automatiquement le processus du jeu et commencera l'analyse
  4. Une fois terminé, un SDK sera généré dans le dossier output/[nom_du_jeu]

Utilisation du Plugin Cheat Engine

Générer une table pour un jeu Source

  1. Lancez le jeu cible (Black Mesa, Half-Life 2, CS:GO, etc.)
  2. Ouvrez Cheat Engine et attachez-le au processus du jeu
  3. Dans la barre de menu de Cheat Engine, cliquez sur "Source Table Generator"
  4. Le plugin détectera automatiquement le jeu en cours d'exécution
  5. Cliquez sur "Scanner" pour rechercher les pointeurs et structures communes
  6. Une fois le scan terminé, cliquez sur "Générer Table" pour créer la table

Questions fréquemment posées

Est-ce que ces outils fonctionnent avec tous les jeux Source ?

Les outils sont conçus pour fonctionner avec la plupart des jeux basés sur le moteur Source et Source 2. Cependant, certains jeux peuvent nécessiter des configurations spécifiques en raison de modifications apportées au moteur.

Est-ce légal d'utiliser ces outils ?

Ces outils sont destinés uniquement à des fins éducatives et de développement de mods légitimes. L'utilisation de ces outils pour développer des cheats ou d'autres logiciels malveillants est strictement interdite et peut entraîner des bannissements de votre compte de jeu.

Comment puis-je ajouter un nouveau jeu ?

Pour le SDK Generator, modifiez le fichier main.cpp et ajoutez une nouvelle entrée dans la table supportedGames. Pour le plugin Cheat Engine, modifiez le fichier SourceTableGenerator.lua et ajoutez une nouvelle entrée dans la table GameProfiles.
